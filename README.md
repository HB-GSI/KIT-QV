LabVIEW Plug and Play instrument driver for
QVision  for IPE-Quench Detection System UNIQD Typ 3410/3420.

Vendor: KIT Institut für Prozessdatenverarbeitung und Elektronik - IPE

The project GIT workflow contains five branches: 
- MainDev: main development line
- master: released versions

LabVIEW 2014 SP1 is currently used for development.
